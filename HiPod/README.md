// HiPod

HiPod is a modern audio sync and file management app for Apple iPod Classic, generic digital audio players (DAPs), and Android-based players. It provides seamless device syncing, advanced file handling, and deep customization for audiophiles and casual users alike.

## Features
- Sync to iPod Classic, generic DAPs, and Android music players
- Flexible file conversion (DSD to PCM, ALAC/FLAC, and more)
- Customizable sync profiles, file naming, and audio tag management
- Retro iTunes-inspired UI option
- Advanced settings for audiophile workflows
